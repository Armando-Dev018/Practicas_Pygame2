## Contenido del Repositorio

Este repositorio contiene los ejercicios y mini-proyectos de las Sesiones 4, 5 y 6, enfocados en conceptos intermedios y avanzados de Pygame.

### Sesión 4: Animación y Control de Tiempo 
* Ejercicios de rebote con velocidad variable.
* Animación de pulsación (crecimiento y decrecimiento).
* Simulación de gravedad y rebote con pérdida de energía.

### Sesión 5: Imágenes y Sprites
* Escalado dinámico de imágenes con el teclad.
* Animación de un sprite desde una hoja de sprites (spritesheet).
* ]**Mini-proyecto:** Nave que rota hacia el ratón y se mueve.

### Sesión 6: Colisiones y Aplicaciones Prácticas
* Detección de colisiones con cambio de color.
* Sistema de recolección de objetos aleatorios con contador.
* Juego de evasión de obstáculos móviles.
* **Mini-proyecto:** Juego simple que combina recolección de puntos y evasión de obstáculos.
